# Deployable Inner-Spring Bronchoscopy Robot

## Overview

This project investigates an extension to a spring-based tendon-driven continuum robot (TDCR) for robotic bronchoscopy.

The proposed design introduces an inner deployable spring that can extend beyond the outer TDCR structure through a threaded distal coupling mechanism. Future work may investigate magnetic steering and advanced control strategies.

## Current Focus

- Literature review
- Spring selection
- Threaded deployment concepts
- Initial CAD development
- Actuation and control concepts
